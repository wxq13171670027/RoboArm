# UR5机器人轨迹设计器
> 本项目是AIS机器人学课程的实践内容之一，旨在帮助学生理解和掌握机器人轨迹规划的基本概念和方法。

这是一个用于UR5机器人轨迹设计和分析的MATLAB应用程序。通过直观的图形界面，用户可以轻松地设计、预览和验证各种类型的机器人运动轨迹。

## 🌟 主要特点

- 📊 直观的图形用户界面
- 🎯 多种轨迹类型支持
- 🔍 实时轨迹预览
- ⚡ 轨迹可行性验证
- 📈 奇异性分析
- 💾 轨迹导出功能

## 🎨 支持的轨迹类型

### 1. 圆形轨迹
- 可调参数：
  - 圆心位置 (X, Y, Z)
  - 半径
- 特点：生成完美的圆形运动路径
- 应用：适用于圆周运动、连续循环操作

### 2. 直线轨迹
- 可调参数：
  - 起点坐标 (X, Y, Z)
  - 终点坐标 (X, Y, Z)
- 特点：生成两点之间的直线路径
- 应用：点对点运动、直线切割/焊接

### 3. 矩形轨迹
- 可调参数：
  - 中心位置 (X, Y, Z)
  - 长度
  - 宽度
- 特点：生成规则的矩形路径
- 应用：矩形区域扫描、边框轮廓运动

### 4. 螺旋线轨迹
- 可调参数：
  - 中心位置 (X, Y, Z)
  - 半径
  - 高度
- 特点：生成三维螺旋上升/下降路径
- 应用：3D打印、螺旋缠绕

### 5. S形轨迹
- 可调参数：
  - 起点坐标 (X, Y, Z)
  - 终点坐标 (X, Y, Z)
- 特点：生成平滑的S形曲线
- 应用：避障运动、平滑过渡

### 6. 波浪形轨迹
- 可调参数：
  - 中心位置 (X, Y, Z)
  - 波长
  - 振幅
- 特点：生成正弦波形状的轨迹
- 应用：周期性运动、波浪形扫描

### 7. 八字形轨迹
- 可调参数：
  - 中心位置 (X, Y, Z)
  - 长度
  - 宽度
- 特点：生成∞形状的连续轨迹
- 应用：连续循环运动、关节灵活性测试

## 🛠️ 主要功能

### 轨迹预览
- 实时3D可视化
- 可调节视角
- 显示采样点和连续路径
- 自动坐标轴缩放

### 轨迹验证
- 检查轨迹点是否在工作空间内
- 验证关节角度是否在限制范围内
- 分析运动可行性

### 奇异性分析
- 检测轨迹中的奇异点
- 计算条件数
- 提供优化建议

### 轨迹导出
- 保存轨迹参数
- 导出轨迹点坐标
- 支持多种文件格式

## 💻 使用方法

1. 启动程序
```matlab
ur5_trajectory_designer
```

2. 选择轨迹类型
- 从下拉菜单选择所需轨迹类型
- 调整相应参数面板中的参数

3. 预览和验证
- 点击"预览轨迹"查看生成的路径
- 使用"验证轨迹可行性"检查运动是否可行
- 点击"分析轨迹"进行深入分析

4. 导出轨迹
- 点击"导出轨迹"保存设计
- 选择保存位置和文件格式

## 🔧 参数调节

### 通用参数
- 采样点数：控制轨迹的精度（范围：10-200）
- 坐标范围：
  - X: [-0.85, 0.85] m
  - Y: [-0.85, 0.85] m
  - Z: [0, 0.85] m

### 可视化控制
- 网格显示
- 坐标轴标签
- 视角调节
- 实时更新
